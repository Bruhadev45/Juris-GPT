Section 01

Project Overview

JurisGPT (formerly NyayaSetu) is an AI-powered legal services platform designed specifically for Indian startups and MSMEs. It combines the power of large language models with structured legal knowledge to provide affordable, accessible, and reliable legal assistance.

The platform operates on an 80% AI + 20% Lawyer hybrid model — AI handles routine legal tasks such as document drafting, compliance tracking, and legal research, while qualified lawyers review critical outputs and handle complex matters that require human judgment.

Section 08

How to Run Locally

Prerequisites

  • Node.js 18+ and npm
  • Python 3.10+
  • Supabase account with project credentials
  • OpenAI API key

Backend Setup

# Navigate to the backend directory
cd backend

# Create and activate a virtual environment
python -m venv venv
source venv/bin/activate          # macOS/Linux
venv\Scripts\activate             # Windows

# Install dependencies
pip install -r requirements.txt

# Configure environment variables
cp .env.example .env
# Edit .env with your Supabase & OpenAI keys

# Start the development server
uvicorn app.main:app --reload

# API available at http://localhost:8000
# Swagger docs at http://localhost:8000/docs

Frontend Setup

# Navigate to the frontend directory
cd frontend

# Install dependencies
npm install

# Configure environment variables
cp .env.example .env.local
# Edit .env.local with your backend URL

# Start the development server
npm run dev

# Visit http://localhost:3000

Environment Variables

Variable Service Description
SUPABASE_URL Backend Supabase project URL
SUPABASE_KEY Backend Supabase anon/service key
OPENAI_API_KEY Backend OpenAI API key for GPT-4o
NEXT_PUBLIC_API_URL Frontend Backend API URL (default: http://localhost:8000)
